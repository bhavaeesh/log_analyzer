## Getting Started
### Backend Setup
To use the backend, install the necessary libraries using pip with the req.txt file.
- cd backend
- pip install -r req.txt

### Frontend Setup
To use the frontend, use yarn to add node modules.
- cd frontend
- yarn install
- yarn start
